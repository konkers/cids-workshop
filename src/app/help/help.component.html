This tracker aims to be an all in one tracker for Final Fantasy IV: Free Enterprise 0.3.0. Currently we support:

  • Characters, key items, and bosses
  • Location and points of interest availability
  • Shops with a configurable set of items
  • Trapped Chests
  • Nk and Nc flags

More features are planned. See our enhancements section on GitHub

For basic character, boss, and key item tracking, chose a location on the left, then click on the character, boss, item at the top. The tracker will record what you found as well as where you found it. As you find key items, more locations and points of interest will unlock. Try selecting the Magma Key as your key item at Into at Baron and see how all the underworld locations unlock.

For more advanced uses (shop items, flags, etc.) check out the actions menu at the top right.

If you find any problems or have feature requests, feel free to file a PR on GitHub or find me on Discord: konkers#0811.
